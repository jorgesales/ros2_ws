# ROS 2 Turtlesim Examples

A collection of Python nodes for ROS 2 Humble designed to control the `turtlesim` simulator. This repository contains various geometric patterns and exercises, from basic movement to complex shapes like the Olympic rings.

## 🚀 Implemented Examples
- **Olympic Rings**: Draws the five Olympic rings using `TeleportAbsolute`, `SetPen`, and `cmd_vel`.
- **Basic Shapes**: (Coming soon) Triangle, Square, and Custom Polygons.
- **Service & Topics**: Examples showing how to mix asynchronous service calls with continuous topic publishing.

## 🛠️ Prerequisites
- **ROS 2 Humble** (or later)
- **Turtlesim** package:
  ```bash
  sudo apt install ros-humble-turtlesim
